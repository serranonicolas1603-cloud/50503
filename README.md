# Analizador Sintáctico con ANTLR4 y JavaScript

## Descripción

Este proyecto implementa un analizador léxico, sintáctico e intérprete básico utilizando ANTLR4 y JavaScript para un subconjunto reducido del lenguaje JavaScript.

El analizador permite:

- Reconocimiento léxico
- Reconocimiento sintáctico
- Generación de tabla de tokens
- Construcción del árbol sintáctico
- Interpretación básica del código

---

## Tecnologías utilizadas

- Node.js
- ANTLR4
- JavaScript
- Visual Studio Code
